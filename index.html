More than 10 years ago e-conomic was founded on the idea to create accounting software in the cloud for Danish businesses. As with any other software 10 years down the road it carries a lot of legacy. Some technologies aren't used any more, code haven't been updated for a while and you wish your architecure/database was different.

Beginning a few years back we made a concious choice to begin using newer technologies, to create an infrastructure that could scale, indepent on the programming languages used. And today we always use the right tool for the job. We have a ton of different services running which can be independently deployed. There is still legacy, and there is still things we wish were different - but we love the challenge and we love to keep working.

As we move forward and gain more experience we open source more and more of our tools and encourage our developers to contribute to other open source projects.
